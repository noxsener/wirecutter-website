Download

To use Codenfast Wirecutter Bot without any problems, please follow the steps below carefully..

1. Java Development Kit (JDK) Installation

For the application to work properly, JDK 17 must be installed on your computer.

If it is not installed, you can download and install JDK 17 from Oracle or OpenJDK's official websites .

2. Download Application

Download the Wirecutter Bot app from: wirecutter-bot.jar

3. Configuring the application.properties File

After downloading, you need to create an application.properties file for the application to work.

Find the example-application.properties file, which is an example of this file, and edit its values to your own configuration. After editing, rename the file to application.properties and save it in the directory where the application was downloaded.

4. First Run of the Application and Database Creation

When you first launch the application, it will create a unique database. During this process, a file named wirecutter.db will be created. It's normal for your computer to consume significant power during this one-time process. The process is complete when the database size reaches approximately 2 GB.

However, you can start processing data from the FTP server without waiting by clicking the "Start Processing" button at the top of the application's interface. Please follow the application's instructions carefully. If you encounter any error messages, please do not hesitate to contact us for assistance.

5. In-Game Bot Use

If the app is working fine, follow these steps to use the in-game bot:

Start the Game: Open the game and go to the main screen.

Character Requirements: You must have a character created on your server with admin rights, and the game language must be set to "English".

Bot Progress: The bot will progress (if not in drone mode) simply by switching to drone mode and pressing the "Continue" button.

In-Game Operation: The bot will automatically open the chat window in-game and set it to GLOBAL. It will then continue to operate, continuously processing data from its own database and the central database.

Automatic Reconnection: If the bot drops out of the game, it will automatically attempt to reconnect.

6. Stopping and Shutting Down the Boat

Stuck Status: If the bot gets stuck, please ask for help.

Closing the Game: You can use the ALT + F4 key combination to close the game (closing it with the mouse may be difficult).

Stopping the Bot: You can stop the bot by clicking the "End-InGame Bot" button in the application interface.

By following these steps, you can successfully install and start using Codenfast Wirecutter Bot. Enjoy!

If you encounter any problems or need further assistance, please let us know.
